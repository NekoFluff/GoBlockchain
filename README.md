## Go Dependencies

```
go get github.com/davecgh/go-spew/spew
go get github.com/gorilla/mux
go get github.com/joho/godotenv
```

**Spew**: Pretty printing for structs and slices<br>
**Mux**: Web handler<br>
**GoDotEnv**: Read .env files<br>
